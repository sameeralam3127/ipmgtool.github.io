# IPMG

**IP Management & Ping Monitoring Tool**

IPMG is a modern, modular, enterprise-ready network scanner and monitoring CLI.

## Who is it for?

- Network Administrators
- DevOps & SREs
- Cybersecurity Teams
- Systems Engineers

## Key Highlights

- Subnet auto-discovery
- Parallel ICMP pinging
- Hostname resolution
- XLSX / CSV / JSON reporting
- Scheduled scans
- Clean CLI interface

> ⚠️ Authorized use only. Do not scan networks without permission.
