# IPMG

**IP Management & Ping Monitoring Tool**

IPMG is a modern, high-performance CLI for managing IP inventories and performing fast, parallel network reachability checks.

---

## Why IPMG?

✔ Designed for real infrastructure  
✔ Safe, explicit, authorized scanning  
✔ CLI-first & automation-ready  
✔ Clean architecture, not scripts

---

## Typical Use Cases

- Validate large IP inventories
- Monitor subnet availability
- Assist network audits
- Support security & compliance teams

> ⚠️ **Authorized use only**. Always obtain explicit permission before scanning.
